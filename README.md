# Gravitas AI: A Multi‑Tool Agent for Modeling General Relativity Scenarios

Gravitas AI is a scientific reasoning agent built using **Google’s Agent Development Kit (ADK) 2.0**, **Antigravity**, and **Gemini models**.  
It transforms natural‑language questions about gravity, black holes, orbits, and spacetime into:

- structured physical scenarios  
- symbolic and numerical calculations  
- orbit and lensing simulations  
- visualization specifications  
- clear, pedagogical explanations  

This project is my submission for the **VibeCoding AI Agents Capstone Project**.

---

## 🚀 Features

### 🧠 Scientific Reasoning
- General relativity (GR) and special relativity (SR)
- Schwarzschild and Newtonian approximations
- Time dilation, redshift, orbital dynamics, lensing

### 🛠️ Multi‑Tool Architecture
- Relativistic math engine  
- Scenario builder  
- Orbit simulator  
- Lensing calculator  
- Physics validator  
- Visualization spec generator  
- Astrophysics constant lookup  
- Explanation formatter  

### 🧩 Skills
- Scenario interpretation  
- GR/SR computation  
- Orbit analysis  
- Lensing analysis  
- Visualization  
- Knowledge retrieval  
- Explanation generation  
- Physics validation
